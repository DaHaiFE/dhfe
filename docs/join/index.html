# 前端架构师

# 职位描述：

  1. 精通Javascript语言，熟练掌握作用域，闭包，执行模型，性能优化，内存管理，网络通信。
  2. 具备实际的大型应用架构能力，清晰的代码架构思想, 高质量实现及合理的分层结构及扩展。
  3. 具备富客户端应用开发经验，至少熟悉一门后台语言，熟悉整体前后端研发流程，较强的整体把控能力。
  4. 熟悉各类常见WEB技术标准规范，流行的前端框架，能够很好的把控到细节层面，例如：React、Vuejs、SVG、CANVAS、WEBGL等。
  5. 具备实际的前端架构，针对业务、团队完全定制化的能力，前端整体工程能力较强。

# 工作职责:

  1. 负责在线教育产品的功能开发、产品优化体验及跨部门项目推动工作。
  2. 负责代码性能、产品技术工程体系建设、技术培训指导其他小伙伴。
  3. 团队内部技术分享、团队内部培养、推动业务、专业目标达成。

# 优先考虑：

  1. 热爱教育行业
  2. 全栈开发能力
  3. 自驱能力较强

# 高级前端开发工程师

# 职位描述：

  1. 4年+以上熟练Javascript语言，对于生态、历史、使用范围、优缺点比较了解。
  2. 扎实的HTML5、CSS3、Javascript3大件基本功, 对于兼容性, 响应式, 移动端、PC 端开发具备丰富的实战经验.
  3. 涉猎多个应用方向，不限于数据展示, 富客户端, WEB网站, 移动混合开发, Electron客户端, 动画交互，游戏等.
  4. 对前端前沿技术较为敏感，学习能力强，用技术推动业务的创新及产品的体验，注重工程能效。
  5. 熟悉常见的HTTP网络协议，缓存，性能优化，安全等问题, 熟悉至少一种后台开发语言(node,php,java,go等), 动手及实践能力较强.

# 工作职责:

  1. 负责在线教育产品的功能开发、产品优化体验及跨部门项目推动工作。
  2. 负责代码性能、产品技术工程体系建设、技术培训指导其他小伙伴。
  3. 研究关注最新前端技术的发展, 用技术改善团队及业务的效率。

# 优先考虑：

  1. 热爱教育行业
  2. 全栈开发能力
  3. 自驱能力较强
