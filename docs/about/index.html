# 我们的风格

不仅仅是工程师，更是团队的建设者。
你可以参与每一个项目，只要你能贡献你的力量。
我们不会很明确地要求你做什么、怎么做，你需要自己思考去做什么、怎么做，给你更大的创新空间。
我们有自己的技术产品和用户，人人都是 PM，需要主动发现并解决遇到的问题，或者提出更好的方案并实现。
